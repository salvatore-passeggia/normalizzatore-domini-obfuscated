## 📋 Descrizione

Strumento web per normalizzare liste di URL e domini. Rimuove protocolli (http://, https://, ftp://), porte, path e parametri, preservando il www quando presente.

## ✨ Funzionalità

• **Normalizzazione** - Pulisce URL e domini mantenendo il www
• **Importa file** - Carica liste da file .txt
• **Esporta risultati** - Salva o copia la lista normalizzata
• **Formato output** - www.example.com (il www viene sempre preservato)
